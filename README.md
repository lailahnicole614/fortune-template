## HTML

-   h1 for header
-   section with

*   p tag for prompt
*   button
*   text input for user question

*   section for
    -   img of 8 ball
    -   p tag to display the randomly selected answer
    -   button to ask again

## Events

-   button click (submit question)

*   hide prompt
*   display the image
*   pick any number between 0 and length of answers

-   button click (ask again)

*   hide 8 ball and answer
*   show prompt
